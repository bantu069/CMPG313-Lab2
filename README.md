## Project Overview
The aim of this project is to apply the K-Means clustering to a dense network topology with hundreds of connected nodes,each with spatial coordinates and a corresponding energy level.
The task emphasizes data-driven analysis instead of visual intuition due to the data set's quantity and complexity.In order to comprehend network topology, energy distribution, and the limits of the K-Means approach, clustering results are analyzed.

## Objectives
-Apply K-Means clustering using a predefined 'KMeans()'function.

-Analyse a dense network topology with over 600 nodes.

-Interpret clustering results based on position and energy distribution.

-Visualise results using 2D network graphs and 3D surface plots.

-Identify the critical nodes with low energy.

-Understand the limitations and confidence behaviour of K-Means clustering.
